Imputing missing values

Calculate and report the total number of missing values in the dataset

sum(is.na(dat$steps))

## [1] 2304

Devise a strategy for filling in all of the missing values in the dataset. Create a new dataset that is equal to the original dataset but with the missing data filled in. Here, I used the mean of each 5 minute interval. For some reason my function messed up the row numbers, so I renumbered them. Note that the date ordering is now different.

z <- split(dat, dat$interval)

new.dat <- do.call(rbind, lapply(z, function(x) {
    x[is.na(x), "steps"] <- mean(x$steps, na.rm = TRUE)
    x
}))

rownames(new.dat) <- c(1:dim(new.dat)[1])
